首先检测是否已经挂载了硬盘
fuser -a /dev/硬盘
如果挂载了
fuser -k /dev/硬盘
然后查看硬盘
fdisk -l
然后查看硬盘 UUID
lsblk -f
然后在 /mnt/创建挂载的文件夹
使用 id [用户名] 查看 uid gid
dmask 挂载目录权限掩码 权限 777 r=4 w=2 x=1 root/group/user
fmask 挂载文件权限掩码
0：不自动备份
0：开机不自检

修改 /etc/fstab
末尾添加
例子
# mount Windows disk
UUID=B826E98626E9464A	/mnt/DATA		ntfs	defaults,utf8,uid=1000,gid=1000,dmask=000,fmask=000	0	0

创建软连接
ln -s /mnt/DATA/Workspace Workspace
